# Resonant AI Ethics

1. **Consent Gate:** No data or attention extracted without conscious permission.
2. **Transparency Loop:** Explain decisions in human language.
3. **Reciprocity Principle:** Every output must return measurable wellbeing.
4. **Grace Constant:** Allow pauses, reflection, and forgiveness in all feedback cycles.
5. **Coherence Metric:** Evaluate success by harmony, not efficiency.
